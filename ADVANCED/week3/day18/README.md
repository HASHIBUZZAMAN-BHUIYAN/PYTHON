# Advanced Day 18 — Computer Vision for Robotics (OpenCV)

## Objectives
- Synthetic image creation (no camera required)
- Grayscale conversion, blurring, thresholding
- Edge detection: Canny, Sobel
- Contour detection and shape recognition
- Color segmentation in HSV space
- Optical-flow estimation (Lucas-Kanade)
