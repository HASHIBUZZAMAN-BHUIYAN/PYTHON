# Day 14 — Capstone: Command-Line Contact Book

## What you'll build
A fully functional CLI contact book that combines **every concept from Days 1–13**:
- Classes & OOP (Day 11/12)
- File handling / CSV persistence (Day 09)
- Error handling (Day 10)
- Functions & modules (Day 08/13)
- Loops, conditionals, dicts, lists, strings (Days 03–07)

## Features
- Add / edit / delete contacts
- Search by name, phone, or tag
- Categorize contacts with tags
- Save to CSV, load on startup
- Input validation with helpful errors

## How to run
```powershell
python BEGINNER\day14\mini_project.py
```
